# 일하는척 (fake-work)

> 업종을 선택하면 해당 직군의 "바빠 보이는" 화면을 풀스크린으로 재생하는 정적 웹앱.
> 런처 화면조차 촌스러운 사내 그룹웨어로 위장되어 있어, 모드를 고르는 순간에도 업무 중으로 보입니다.

**본 시스템은 실제 업무 성과를 생성하지 않습니다. 성과에 대한 책임은 전적으로 사용자에게 있습니다.**

## 모드

| 모드 | 화면 | 자동 재생 |
|---|---|---|
| 💻 개발자 | Windows Terminal (WSL) | 빌드/테스트/k8s 로그 스트림 + 해커타이퍼 |
| 🛢️ 데이터 엔지니어 | airflow-worker 셸 | Airflow·Debezium·dbt·Spark 로그 + 해커타이퍼 |
| 📊 사무직 | Excel (`v7_최종_진짜최종.xlsx`) | 셀이 저절로 채워지고 수식바가 갱신됨 |
| 📈 마케터 | 밝은 SaaS 대시보드 | KPI 실시간 변동 + 흐르는 라인차트 |
| 📉 트레이더 | 다크 HTS | 0.26초 간격 체결 플래시, 지수 변동 |
| 📝 보고서 | Word A4 문서 | 「추진 계획(안)」이 완벽한 보고서체로 자동 타이핑 |
| 🎨 디자이너 | Figma 캔버스 | 시안이 저절로 다듬어지고 팀원 커서가 떠다님 |
| 🎧 상담원 | 헬프데스크 | 티켓이 쌓이고 처리되는 실시간 큐 |
| 📋 기획자 | Jira풍 칸반보드 | 카드가 저절로 흘러가고 가끔 반려됨 |
| 📚 연구원 | 논문 PDF 뷰어 | 형광펜 하이라이트가 저절로 그어짐 |

## 사용법

- 런처에서 카드를 클릭(또는 Tab + Enter)하면 모드 진입
- **ESC**: 어느 화면에서든 즉시 그룹웨어 런처로 복귀 (긴급 탈출)
- 터미널 모드에서 아무 키나 누르면 코드가 타이핑됩니다
- `#/dev` 처럼 URL 해시로 직접 진입 가능 — 새로고침해도 모드가 유지됩니다

## 개발

```bash
npm install
npm run dev      # 개발 서버
npm test         # 유틸/스케줄러 단위 테스트
npm run build    # 타입체크 + 프로덕션 빌드
```

외부 런타임 의존성 0개 (devDependencies만 사용), 프레임워크 없는 vanilla TypeScript + Vite.
아키텍처와 모드 추가 방법은 [CLAUDE.md](./CLAUDE.md) 참고 — 모드 추가는 폴더 1개 + import 1줄로 끝납니다.

## 배포

`main` 브랜치에 push하면 GitHub Actions가 GitHub Pages로 자동 배포합니다.
저장소 설정에서 **Settings → Pages → Source: GitHub Actions**를 한 번 켜 주세요.
빌드 base 경로가 `/fake-work/`이므로 저장소 이름은 `fake-work`여야 합니다.
